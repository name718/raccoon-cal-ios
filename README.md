# 浣熊卡路里 (RaccoonCal)

一个基于iOS原生技术的智能卡路里管理应用，通过拍照识别食物卡路里，结合虚拟宠物养成和社交功能，让健康管理变得有趣。

## 🦝 项目特色

- **智能识别**：拍照即可识别食物并计算卡路里
- **虚拟宠物**：通过健康饮食养成可爱的浣熊宠物
- **社交互动**：与朋友分享健康生活，互相监督鼓励
- **原生体验**：使用Swift + SwiftUI构建，性能优异

## 🛠 技术栈

- **前端**：iOS Native (Swift + SwiftUI)
- **后端**：Express.js + Node.js
- **数据库**：MongoDB + Redis
- **包管理**：Swift Package Manager

## 📱 功能模块

1. **拍照识别卡路里**
   - 智能食物识别
   - 营养成分分析
   - 卡路里计算

2. **虚拟宠物养成**
   - 浣熊宠物系统
   - 健康行为奖励
   - 宠物成长机制

3. **社交功能**
   - 好友系统
   - 健康挑战
   - 成就分享

## 🚀 快速开始

### 环境要求

- Xcode 14.0+
- iOS 15.0+
- Swift 5.7+

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/name718/raccoon-cal.git
cd raccoon-cal
```

2. 打开Xcode项目
```bash
open RaccoonCal.xcodeproj
```

3. 运行项目
- 选择目标设备或模拟器
- 点击运行按钮或按 `Cmd + R`
